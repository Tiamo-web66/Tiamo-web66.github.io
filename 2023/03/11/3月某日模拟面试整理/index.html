    3. иҪ¬жҚўе’ҢдјҳеҢ–пјҡжү“еҢ…е·Ҙе…·еҸҜд»ҘеҜ№д»Јз ҒиҝӣиЎҢиҪ¬жҚўе’ҢдјҳеҢ–пјҢжҜ”еҰӮеҺӢзј© JavaScript е’Ң CSS д»Јз ҒпјҢдјҳеҢ–еӣҫзүҮеӨ§е°ҸзӯүпјҢеҮҸе°Ҹж–Үд»¶дҪ“з§ҜпјҢжҸҗй«ҳеҠ иҪҪйҖҹеәҰгҖӮ
    4. ж”ҜжҢҒеӨҡз§Қж јејҸпјҡжү“еҢ…е·Ҙе…·еҸҜд»Ҙе°ҶдёҚеҗҢж јејҸзҡ„ж–Үд»¶жү“еҢ…жҲҗдёҖдёӘйҖҡз”Ёзҡ„ж јејҸпјҢд»ҘдҫҝдәҺеңЁжөҸи§ҲеҷЁдёӯеҠ иҪҪе’ҢиҝҗиЎҢгҖӮ

    з»јдёҠжүҖиҝ°пјҢеүҚз«Ҝжү“еҢ…еҸҜд»ҘжҸҗй«ҳйЎөйқўжҖ§иғҪе’ҢејҖеҸ‘ж•ҲзҺҮпјҢжҳҜеүҚз«ҜејҖеҸ‘дёӯдёҚеҸҜжҲ–зјәзҡ„дёҖзҺҜгҖӮ

  • еҰӮдҪ•еӨ„зҗҶи·ЁеҹҹиҜ·жұӮ

    и·ЁеҹҹиҜ·жұӮжҳҜжҢҮеңЁжөҸи§ҲеҷЁдёӯеҗ‘дёҚеҗҢжәҗзҡ„жңҚеҠЎеҷЁеҸ‘йҖҒиҜ·жұӮгҖӮз”ұдәҺжөҸи§ҲеҷЁзҡ„еҗҢжәҗзӯ–з•ҘйҷҗеҲ¶пјҢи·ЁеҹҹиҜ·жұӮеҸҜиғҪдјҡиў«жөҸи§ҲеҷЁжӢ’з»қгҖӮд»ҘдёӢжҳҜдёҖдәӣеӨ„зҗҶи·ЁеҹҹиҜ·жұӮзҡ„ж–№жі•пјҡ

    • JSONP

    JSONPжҳҜдёҖз§ҚеҲ©з”Ё<script>ж Үзӯҫзҡ„и·Ёеҹҹж–№жі•гҖӮе®ўжҲ·з«ҜйҖҡиҝҮеңЁURLдёӯжҢҮе®ҡеӣһи°ғеҮҪж•°зҡ„еҗҚз§°пјҢжңҚеҠЎеҷЁиҝ”еӣһдёҖдёӘJavaScriptеҮҪж•°и°ғз”ЁпјҢиҜҘеҮҪж•°зҡ„еҸӮж•°е°ұжҳҜжңҚеҠЎеҷЁиҝ”еӣһзҡ„ж•°жҚ®гҖӮз”ұдәҺ<script>ж ҮзӯҫдёҚеҸ—еҗҢжәҗзӯ–з•ҘйҷҗеҲ¶пјҢеӣ жӯӨеҸҜд»Ҙи·Ёеҹҹи°ғз”ЁжңҚеҠЎеҷЁз«Ҝзҡ„и„ҡжң¬е№¶иҺ·еҸ–ж•°жҚ®гҖӮдҪҶжҳҜпјҢJSONPеҸӘж”ҜжҢҒGETиҜ·жұӮпјҢдё”еҸӘж”ҜжҢҒзәҜж–Үжң¬ж•°жҚ®пјҢдёҚж”ҜжҢҒдәҢиҝӣеҲ¶ж•°жҚ®е’Ңе…¶д»–ж јејҸзҡ„ж•°жҚ®гҖӮ

    • CORS

    и·Ёеҹҹиө„жәҗе…ұдә«пјҲCORSпјүжҳҜдёҖз§ҚеңЁжңҚеҠЎз«Ҝи®ҫзҪ®зҡ„ж–№ејҸпјҢе…Ғи®ёе®ўжҲ·з«Ҝи·ЁеҹҹиҜ·жұӮжңҚеҠЎеҷЁдёҠзҡ„иө„жәҗгҖӮйҖҡиҝҮеңЁжңҚеҠЎеҷЁз«Ҝи®ҫзҪ®зӣёеә”зҡ„е“Қеә”еӨҙпјҢжҢҮе®ҡе…Ғи®ёе“ӘдәӣеҹҹеҗҚиҝӣиЎҢи®ҝй—®пјҢеҚіеҸҜе®һзҺ°и·ЁеҹҹиҜ·жұӮгҖӮCORSж”ҜжҢҒеҗ„з§ҚHTTPиҜ·жұӮпјҢеҢ…жӢ¬GETгҖҒPOSTгҖҒPUTзӯүпјҢе№¶дё”ж”ҜжҢҒеҗ„з§Қж•°жҚ®ж јејҸпјҢеҰӮJSONгҖҒXMLзӯүгҖӮ

    • д»ЈзҗҶжңҚеҠЎеҷЁ

    д»ЈзҗҶжңҚеҠЎеҷЁжҳҜдёҖз§Қе°Ҷе®ўжҲ·з«ҜиҜ·жұӮиҪ¬еҸ‘еҲ°зӣ®ж ҮжңҚеҠЎеҷЁзҡ„дёӯй—ҙжңҚеҠЎеҷЁгҖӮе®ўжҲ·з«Ҝеҗ‘д»ЈзҗҶжңҚеҠЎеҷЁеҸ‘еҮәиҜ·жұӮпјҢд»ЈзҗҶжңҚеҠЎеҷЁеҶҚе°ҶиҜ·жұӮиҪ¬еҸ‘еҲ°зӣ®ж ҮжңҚеҠЎеҷЁгҖӮз”ұдәҺиҜ·жұӮжҳҜеңЁеҗҢдёҖдёӘеҹҹеҗҚдёӢеҸ‘еҮәзҡ„пјҢеӣ жӯӨдёҚдјҡйҒҮеҲ°и·Ёеҹҹй—®йўҳгҖӮд»ЈзҗҶжңҚеҠЎеҷЁеҸҜд»ҘеңЁе®ўжҲ·з«Ҝе’Ңзӣ®ж ҮжңҚеҠЎеҷЁд№Ӣй—ҙиҝӣиЎҢж•°жҚ®иҪ¬жҚўпјҢе®һзҺ°еҚҸи®®иҪ¬жҚўгҖҒж•°жҚ®ж јејҸиҪ¬жҚўзӯүеҠҹиғҪгҖӮ

  • JSONPзҡ„еҺҹзҗҶ

    JSONPпјҲJSON with PaddingпјүжҳҜдёҖз§Қи·ЁеҹҹиҜ·жұӮзҡ„ж–№ејҸпјҢе®ғеҲ©з”ЁдәҶжөҸи§ҲеҷЁеҸҜд»Ҙи·Ёеҹҹеј•з”ЁдёҚеҗҢеҹҹеҗҚдёӢзҡ„ JavaScript ж–Үд»¶зҡ„зү№жҖ§жқҘе®һзҺ°и·ЁеҹҹиҜ·жұӮгҖӮдёӢйқўжҳҜ JSONP иҝӣиЎҢи·ЁеҹҹиҜ·жұӮзҡ„еҹәжң¬жӯҘйӘӨпјҡ

    1. еңЁе®ўжҲ·з«ҜеҲӣе»әдёҖдёӘ script ж ҮзӯҫпјҢе№¶е°Ҷе…¶ src еұһжҖ§и®ҫзҪ®дёәзӣ®ж ҮжҺҘеҸЈзҡ„ URLпјҢеҗҢж—¶еңЁ URL дёӯдј е…ҘдёҖдёӘеӣһи°ғеҮҪж•°еҗҚпјҢдҫӢеҰӮпјҡ

    2. зӣ®ж ҮжҺҘеҸЈеңЁжҺҘж”¶еҲ°иҜ·жұӮеҗҺпјҢе°ҶиҜ·жұӮеҸӮж•°дёӯзҡ„еӣһи°ғеҮҪж•°еҗҚдҪңдёәеҸӮж•°пјҢжһ„йҖ еҮәдёҖдёӘ JSON ж•°жҚ®пјҢе№¶е°ҶиҜҘж•°жҚ®дҪңдёә JavaScript д»Јз Ғиҝ”еӣһпјҢдҫӢеҰӮпјҡ

    3. е®ўжҲ·з«Ҝе®ҡд№үеӣһи°ғеҮҪж•°жқҘеӨ„зҗҶиҝ”еӣһзҡ„ж•°жҚ®пјҢдҫӢеҰӮ

      з”ұдәҺ JSONP еҲ©з”ЁдәҶжөҸи§ҲеҷЁеҸҜд»Ҙи·Ёеҹҹеј•з”Ё JavaScript ж–Үд»¶зҡ„зү№жҖ§пјҢеӣ жӯӨеңЁдҪҝз”Ё JSONP иҝӣиЎҢи·ЁеҹҹиҜ·жұӮж—¶пјҢйңҖиҰҒзЎ®дҝқзӣ®ж ҮжҺҘеҸЈиҝ”еӣһзҡ„ж•°жҚ®жҳҜдёҖж®өеҸҜжү§иЎҢзҡ„ JavaScript д»Јз ҒгҖӮеҗҢж—¶пјҢз”ұдәҺ JSONP иҜ·жұӮжҳҜйҖҡиҝҮеҠЁжҖҒеҲӣе»ә script ж Үзӯҫе®һзҺ°зҡ„пјҢеӣ жӯӨж— жі•еҸ‘йҖҒ POST иҜ·жұӮгҖӮ

  • д»Җд№Ҳжғ…еҶөдёӢеҜјиҮҙи·Ёеҹҹй—®йўҳ

    и·Ёеҹҹй—®йўҳжҢҮзҡ„жҳҜеңЁзҪ‘йЎөдёӯйҖҡиҝҮ JavaScript еҸ‘йҖҒ HTTP иҜ·жұӮж—¶пјҢеҰӮжһңиҜҘиҜ·жұӮзҡ„еҹҹеҗҚдёҺеҪ“еүҚзҪ‘йЎөзҡ„еҹҹеҗҚдёҚдёҖиҮҙпјҢе°ұдјҡи§ҰеҸ‘и·Ёеҹҹй—®йўҳгҖӮ

    еҪ“е®ғдёҚдёҖиҮҙзҡ„ж—¶еҖҷ,еҸ‘йҖҒзҡ„иҜ·жұӮиў«жөҸи§ҲеҷЁжӢ’з»қжҺҘж”¶(дёҚжҳҜдёҚеҸ‘иҜ·жұӮ),

    и·ЁеҹҹиҜ·жұӮеҲҶдёәдёӨз§ҚиҜ·жұӮ,дёҖз§ҚжҳҜз®ҖеҚ•иҜ·жұӮ,дёҖз§ҚжҳҜйқһз®ҖеҚ•иҜ·жұӮ

    иҝҷжҳҜз”ұжөҸи§ҲеҷЁзҡ„еҗҢжәҗзӯ–з•ҘжүҖеј•иө·зҡ„гҖӮ

    еҗҢжәҗзӯ–з•ҘжҢҮзҡ„жҳҜпјҢдёҖдёӘзҪ‘з«ҷзҡ„и„ҡжң¬еҸӘиғҪи®ҝй—®жқҘиҮӘдәҺеҗҢдёҖзҪ‘з«ҷзҡ„иө„жәҗпјҢиҖҢдёҚиғҪзӣҙжҺҘи®ҝй—®жқҘиҮӘдәҺе…¶е®ғзҪ‘з«ҷзҡ„иө„жәҗгҖӮеҗҢжәҗзӯ–з•ҘжҳҜдёәдәҶдҝқиҜҒзҪ‘з»ңе®үе…ЁиҖҢеӯҳеңЁзҡ„пјҢйҳІжӯўжҒ¶ж„Ҹд»Јз Ғд»ҺдёҖдёӘзҪ‘з«ҷдёӯиҺ·еҸ–еҲ°з”ЁжҲ·зҡ„ж•Ҹж„ҹдҝЎжҒҜпјҢдҫӢеҰӮ cookie зӯүгҖӮ

    дёӢйқўжҳҜеҜјиҮҙи·Ёеҹҹй—®йўҳзҡ„еҮ з§Қжғ…еҶөпјҡ

    1. дёҚеҗҢеҹҹеҗҚпјҡдҫӢеҰӮд»Һ http://example.com еҗ‘ http://api.example.net еҸ‘йҖҒиҜ·жұӮгҖӮ
    2. дёҚеҗҢз«ҜеҸЈеҸ·пјҡдҫӢеҰӮд»Һ http://example.com:80 еҗ‘ http://example.com:8080 еҸ‘йҖҒиҜ·жұӮгҖӮ
    3. дёҚеҗҢеҚҸи®®пјҡдҫӢеҰӮд»Һ http://example.com еҗ‘ https://example.com еҸ‘йҖҒиҜ·жұӮгҖӮ

    еҪ“ JavaScript д»Јз Ғеҗ‘и·Ёеҹҹзҡ„иө„жәҗеҸ‘йҖҒиҜ·жұӮж—¶пјҢжөҸи§ҲеҷЁдјҡеҸ‘еҮәдёҖдёӘ OPTIONS иҜ·жұӮпјҢиҜ·жұӮжңҚеҠЎеҷЁиҝ”еӣһдёҖдёӘ CORS еӨҙйғЁдҝЎжҒҜпјҢе‘ҠиҜүжөҸи§ҲеҷЁиҜҘи·ЁеҹҹиҜ·жұӮжҳҜеҗҰиў«е…Ғи®ёгҖӮеҰӮжһңжңҚеҠЎеҷЁиҝ”еӣһзҡ„ CORS еӨҙйғЁдҝЎжҒҜдёҚе…Ғи®ёиҜҘиҜ·жұӮпјҢеҲҷжөҸи§ҲеҷЁдјҡйҳ»жӯўиҜҘиҜ·жұӮ

      еҫ®д»»еҠЎпјҲmicro taskпјүжҳҜжҢҮеңЁеҪ“еүҚд»»еҠЎжү§иЎҢе®ҢжҜ•еҗҺпјҢйңҖиҰҒз«ӢеҚіжү§иЎҢзҡ„д»»еҠЎгҖӮдҫӢеҰӮ Promise зҡ„ then/catch еҮҪж•°гҖҒMutationObserver зҡ„еӣһи°ғеҮҪж•°зӯүгҖӮеҪ“ејӮжӯҘд»Јз Ғе®ҢжҲҗеҗҺпјҢдјҡиў«ж·»еҠ еҲ°еҫ®д»»еҠЎйҳҹеҲ—дёӯпјҢзӯүеҫ…еҪ“еүҚе®Ҹд»»еҠЎжү§иЎҢе®ҢжҜ•еҗҺз«ӢеҚіжү§иЎҢгҖӮ
