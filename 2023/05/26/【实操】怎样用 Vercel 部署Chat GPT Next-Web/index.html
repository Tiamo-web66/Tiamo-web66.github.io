【实操】怎样用 Vercel 部署Chat GPT Next-Web

演示网址：ai.xiaoadai.com
密码：tiamo666

🚀一、Fork 项目并部署

如果你已经拥有 Github 账号，那么 Fork 项目非常简单，直接点开下方 ChatGPT-Next-Web 的项目地址，然后点击右上角第二个 Fork 按钮即可：

https://github.com/Yidadaa/ChatGPT-Next-Web

这个步骤会将作者整个项目复制到你自己的仓库（Repository），方便后续进行部署等操作。如果你需要后续自动更新，请不要直接下载项目，或者点项目下面的 Deploy 按钮，否则会导致自动更新不可用。

🌟二、使用 Vercel 一键部署

将项目fork到自己仓库后，就可以使用 Vercel 进行一键部署了，但是 Vercel目前不能在国内直接访问，需要一点点魔法技术，请自备尝试，直到能访问 Vercel 为止：

https://vercel.com/signup

如果你还没有 Vercel 账号，可以参考 Vercel 注册教程进行注册，或者你有一个使用较久的 Github账号，也可以直接用 Github 登录使用。

01 添加项目

登陆后，点击右上角的 Add New，添加新项目：

然后按照提示连接到Github，选择你需要导入的项目，然后点击 Import 导入 ChatGPT-Next-Web 项目即可。

然后选择你需要导入 Github项目，这里可以选择所有的，也可以之选择 ChatGPT-Next-Web，选好之后，参考上图点击 Import。

随后在这里填入环境变量，项目目前共有四个环境变量，只有 API Key 是必填项目:

1 环境变量 变量示例 变量说明
2 OPENAI_API_KEY sk_XXXXXXXX 必填 OpanAI密钥，你在openai账户页面申请的apikey。必填项，注意包含sk-
3 CODE password1，password2，password3 可选 访问密码，用英文逗号隔开多个密码。警告：如果不填写此项，则任何人都可以直接使用你部署后的网站，可能会导致你的token被急速消耗完毕，建议填写此选项。
4 BASE_URL api.openai.com 可选，OpenAI 接口代理URL，如果你手动配置了openai接口代理，请填写此选项。可选，默认值魏api.openai.com
5 PROTOCOL https 可选Openai代理接口协议，如果遇到ssl证书问题，请尝试通过此选项设置魏http。可选。默认值为https。可选值为http或https。

填完之后，点击 Deploy按钮，耐心等待安装完成即可

完成后会出现撒花画面，点击 Ccontinue to DashBoard 继续即可。

然后点击 Visit按钮，或者下方域名，即可在魔法环境下访问了，够简单吧！

02 配置域名

安装完成后，项目会自动分配一个 vercel域名地址方便访问，但是由于 vercel.app 域名在国内被污染，通常建议绑定一个自己的域名用作中转，实现国内直连。比如腾讯云的某些域名，就只要四块钱一年：

域名注册请看域名注册篇(还没写），在这里假定你已经拥有一个域名，比如叫 chatgptnextweb.com，那么我们进入 vercel 项目的控制台，通常点击自己头像，再点开项目就能看到：

然后找到 Settings，下方找到 Domains，添加你的域名：

添加后，Vercel会告诉你要将域名解析指向 Vercel的服务器，按提示到域名注册商那边添加解析即可

这里我使用的是腾讯云，就用腾讯云做演示，其他的域名服务商也类似，先登录腾讯云，然后搜域名注册，点击我的域名，然后点击你要解析的域名-解析：

然后我们在这里添加记录：

这里要注意，我这里的主机记录填了 chat 是因为我在 Vercel 绑定的域名叫 chat.chatgptnextweb.com，如果你绑定的域名叫 nextweb.com，这里应该填写 @，如果绑定的是 www.nexweb.com,这里应该填www。好懂吧？

然后我们返回 Vercel，这里会自动刷新，并生成 SSL证书，等证书生成好之后，就可以直连访问啦！

03 Vercel常见问题

Question：KEY 和 Code 在哪里改？

Answer：依然是在你项目中的 Settings-Environment Variables 中修改（找不到 Setting？点左上角头像-点开ChatGPT-Next-Web项目，才能看到 Settings）。改完了还没完事，要重新部署一遍才能生效！

这里改完之后，到 Deployments 点击最新的一个部署， Promote to Production，才能将最新的环境变量生效。

Question：之前部署错了，怎么删掉项目？

Answer：依然是在 Settings里面，General翻到底部，点击 Delete即可，Vercel会让你手动输入删除直连（黑色加粗字体），嫌麻烦也可以复制粘贴

Question：怎么升级到最新版本

Answer：Vercel 支持自动同步 Github 仓库，因此，你在 Github 仓库中的任何改动，都会导致 Vercel重新部署，你要做的就是在你自己仓库里的项目中给，Sync fork 即可，稍等两三分钟就能升级到最新版本。

Question：怎么改界面内容

Answer：这是一篇小白教学，这个操作稍微复杂了点，但也不是不能教。如果要修改主界面的 ChatGPT Next，直接在的 Github仓库中搜索这段文字，然后去自己的Github仓库改就完事了，具体怎么改，有空单独出教程。
