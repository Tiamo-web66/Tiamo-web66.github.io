既然使用Python，那么我们有很多的库进行选择，那么我们首先确定库使用什么比较好。

既然我们引入了这些库，那么我们需要了解这些都是干什么的。

这些库的作用如下：

  1. urllib3: urllib3 是一个功能强大的 HTTP 客户端库，用于发送 HTTP 请求。它提供了连接池管理、重试、超时等功能，使得发送 HTTP 请求更加方便和灵活。

  2. os: os 是 Python 的内置库，提供了与操作系统交互的功能。它可以用于执行文件和目录操作，例如创建、删除、重命名文件或目录，获取文件属性等。

  3. time: time 是 Python 的内置库，提供了与时间相关的功能。它可以用于获取当前时间、计时等操作。在代码中，time 库可能用于添加时间戳、计时等功能。

  4. requests: requests 是一个常用的第三方库，用于发送 HTTP 请求。它提供了简洁的 API 接口，使得发送 HTTP 请求变得更加简单和易用。通过 requests 库，可以发送 GET、POST 等各种类型的请求，并处理响应结果。

  5. datetime: datetime 是 Python 的内置库，提供了日期和时间的处理功能。它可以用于获取当前日期和时间、格式化日期和时间、进行日期和时间的计算等操作。在代码中，datetime 库可能用于处理日期和时间相关的逻辑。

  6. hashlib: hashlib 是 Python 的内置库，提供了各种哈希算法的实现。它可以用于计算文件或数据的哈希值，例如 MD5、SHA1 等。在代码中，hashlib 库可能用于计算文件的哈希值或进行数据的完整性校验。

这些库提供了丰富的功能，可以帮助我们实现各种与网络请求、文件操作、时间处理和数据校验相关的任务。

首先我们要进行禁用警告信息的输出，urllib3.disable_warnings() 是 urllib3 库提供的一个函数调用，用于禁用警告信息的输出。

在发送 HTTP 请求时，有些网站可能使用了自签名的 SSL 证书或其他不受信任的证书，这会导致 urllib3 库默认会输出警告信息，提醒用户存在潜在的安全风险。但在某些情况下，我们可能希望忽略这些警告，例如在测试环境中或者确保自己信任网站的情况下。

调用 urllib3.disable_warnings() 函数可以禁用 urllib3 库的警告信息输出。这样，当使用 urllib3 发送请求时，即使遇到不受信任的证书，也不会产生警告信息。

需要注意的是，禁用警告信息可能会带来一定的安全风险，请谨慎使用，并确保你对目标网站的信任和了解。
